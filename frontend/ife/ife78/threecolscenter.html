三栏布局，左右定宽中间自适应

左右double-float

float: left
float: right
margin-left: 140px; margin-right: 140px;
缺点：中间内容无法首先加载

左右double-float+BFC

float: left
float: right
display: table-cell; width: 9999px; *display: inline-block; *width: auto;
缺点：中间内容无法首先加载
优点：不用计算margin,中间设置margin无效

双飞翼

content: float: left;width: 100%;
center:margin-left: 140px;margin-right: 140px;
缺点：HTML代码复杂提高
优点：中间内容可以优先显示
margin-left:-100%;
margin-left:-120px;

圣杯

content: margin-left: 140px;margin-right: 140px;
center:width: 100%;float:left;
缺点：CSS代码复杂提高
优点：中间内容可以优先显示
position: relative; left: -140px;
position: relative; right:-140px

flex

flex: 1 1 auto;
缺点：浏览器兼容
order: -1;flex: 0 1 120px;
flex: 0 1 120px;

table

order: -1;flex: 0 1 120px;
flex: 1 1 auto;
flex: 0 1 120px;

absolute+margin

margin: 0 140px;
优点：简单，主要内容优先加载
缺点：绝对元素内容过高会出现父元素高度溢出 缺点：浏览器兼容
left: 0;top:0;position: absolute;
left: 0;top:0;position: absolute;
left: 0;top:0;position: absolute;
right: 0;top:0;position: absolute;
